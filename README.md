# PokePrep

The goal of this web application is to make the process of planning and preparing teams for draft leagues more streamlined. As it stands, when I make a team for a draft league I usually have 4 tabs open:
- Pokemon Showdown (the team editor)
- Smogon
- A Google Sheets form that has type and speed matchups
- Pokemon Damage Calculator

This web application combines the team editor, google sheets form, and the damage calculator into one handy tool. 

## Damage Calculator

Fully functioning damage calculator, akin to https://calc.pokemonshowdown.com/. Will not have Smogon datasets, but could in the future.

## Team Editor

Simple and intuitive team editor that updates automatically and can be exported at the touch of a button. 

## Type Matchup

Easy-to-understand type matchup for both your team and the opponent's

## Speed Matchup

Easy-to-understand, sorted speed matchup for both teams
